# 🍕 Ñam Ñam Pizzería — Proyecto Web

## Cómo abrir este proyecto en Cursor

1. **Descarga** el archivo `namnam-pizzeria.html` desde esta conversación
2. **Crea una carpeta** en tu ordenador llamada `namnam-web`
3. **Mueve** el archivo `namnam-pizzeria.html` dentro de esa carpeta y **renómbralo** a `index.html`
4. **Abre Cursor** → File → Open Folder → selecciona la carpeta `namnam-web`
5. **Abre el chat de Cursor** (Ctrl+L o Cmd+L) y pega el prompt de abajo
